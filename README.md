## Usage

<a name="setup"></a>

### Setup

`mvn clean install`

<a name="run"></a>

### Run

`mvn exec:java`

<a name="build"></a>

### Build

`mvn package`

Outputs to `target/algorithm-analyzer-x.x.x.jar`

<a name="test"></a>

### Test

`mvn test`

<a name="formatting-code"></a>

### Formatting Code

`/bin/sh bin/format.sh`

Calls google-code-formatter locally.
